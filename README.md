# bayes-method
Coursework for AMS 522 Bayesian Methods in Finance at Stony Brook University, mainly use python and Matlab.

Includes small projects such as：
-Black–Litterman allocation,
-copula fitting with mixed loss, 
-CVaR portfolio optimization, 
-CVaR regression & tail risk, 
-entropy pooling,
-a simple comparison between Bayesian and MLE estimation.
